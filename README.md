# MIni-Assistente
Notebook em Jupyter que simula a triagem de mensagens de um escritório jurídico em tempo real. Você digita uma mensagem (estilo WhatsApp) e ele chama uma LLM via OpenRouter (Llama 3.3 70B) para retornar um JSON mínimo com classe (Processual, Financeiro etc) e uma justificativa curta.
